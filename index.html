WebAudioXML Sonification ToolKit

s. Data Range: Parameter Range:

The data comes from https://www.gapminder.org/data/ and shows the CO2-emissions per person from 1800 in different countries and estimated to 2100.

Parameter Mapping:

  1. Select a data source
  2. Select and Audio Object
  3. Select a parameter
  4. Adjust the ranges
  5. Play or scrub to listen
  6. Add more parameters and/or data sources

Configure data and sound model:

  1. Replace the data with your own by updating data.csv-file (follow the provided data-structure)
  2. Replace the audio configuration by editing the audio.xml-file (https://github.com/hanslindetorp/WebAudioXML)
